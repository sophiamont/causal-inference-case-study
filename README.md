# causal-inference-case-study

Causal inference playground focused on a synthetic control case study.

## Setup

```bash
uv sync
```

Or install from the frozen environment:

```bash
pip install -r requirements.txt
```

## Project structure

```text
causal-inference-case-study/
├── data/
│   └── raw/
├── notebooks/
│   └── 01_synthetic_control.ipynb
├── README.md
└── requirements.txt
```
